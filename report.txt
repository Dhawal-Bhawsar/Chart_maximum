Frontend Intern Assignment
Build a Custom Chart Component (Angular)
Objective
Create a reusable Angular chart component that renders different chart types based on configuration input. This
assignment evaluates HTML, CSS, JavaScript, Angular usage, and component design.
Task Description
Develop a custom Angular component: <io-chart [chartOptions]="options"></io-chart> The component must
render charts dynamically based on input.
Chart Options Format
interface ChartOptions { type: 'line' | 'column' | 'pie'; title: string; series: { name: string; value: number; color: string;
}[]; }
Example Input
chartOptions = { type: 'line', title: 'Sales Report', series: [ { name: 'Offline', value: 30, color: 'red' }, { name: 'Online',
value: 70, color: 'blue' } ] };
Supported Chart Types
1. Line Chart – Connected points with axes. 2. Column Chart – Vertical bars. 3. Pie Chart – Circular segments.
Technical Rules
Allowed: - Angular components - HTML/CSS/SVG/Canvas - TypeScript Not Allowed: - External chart libraries -
Copy-paste packages
